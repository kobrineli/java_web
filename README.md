# Описание страниц
--------------
Сверху для всех пользователей (зарегистрированных, незарегистрированных и
администраторов) в web-пирложении располагается панель с кнопками, по которым
можно перейти на соответствующие страницы, далее будет
называться верхним меню.

## Для незарегистрированных пользователей
### Кнопки в верхнем меню
1. Кнопка Main page
2. Кнопка Registration
3. Кнопка Log in

### 1. Main page
Является главной страницей, на которой расположен список книг. Список состоит из
гиперссылок, содержащих название и автора (авторов). При нажатии на гиперссылку
осуществляется переход на страницу конкретной книги. Также на главной странице
доступна фильтрация списка книг по названию, автору, кол-ву экземлпяров, кол-ву
свободных экземпляров, издательству и году издания.

### 2. Registration
Страница регистрации с формой для регистрации и заполнения данных нового
читателя.

### 3. Log in
Страница входа пользователя на сайт под свои логином и паролем.

### Страница книги
Страница со всей информацией о книге: название, автор, кол-во экземпляров,
кол-во свободных экземпляров, издательство, год издания.

## Для зарегистрированных пользователей
### Кнопки в верхнем меню
1. Кнопка Main page
2. Кнопка Profile
3. Log out
4. History

### 1. Main page
Аналогично такой же странице для незарегистрированных пользователей: список книг
с возможностью фильтрации и переходом на страницу конкретной книги.

### 2. Profile
Страница с личной информацией пользователя: логин, пароль, ФИО, адрес
проживания. номер телефона, номер читательского билета.
Доступна кнопка Edit - переход на страницу редактирования личной информации.

### 3. Log out
Переход на главную страницу незарегистрированного пользователя.

### 4. History
страница с историей прочитанных читателем книг с датой, когда читатель взял
книгу, и когда читатель вернул книгу (может быть пустым).
Если читатель еще не взял книгу, присутствует кнопка отмены бронирования книги.

### Страница книги
Аналогично странице книги для незарегистрированного читателя, но теперь доступна
кнопка бронирования книги.

### Бронирование книги
Переход на эту страницу осуществляется со страницы конкретной книги путем
нажатия на кнопку бронирования книги. Заполняется поле даты, когда читатель
планирует забрать книгу.

## Для администратора
### Кнопки в верхнем меню
1. Кнопка Main admin page
2. Кнопка Orders
3. Кнопка Readers
4. Кнопка Add book
5. Кнопка Add reader
6. Кнопка Log out

### 1. Main page
Главная страница со списком книг, аналогичная главной странице обычных
пользователей (см. выше).

### 2. Orders
Страница для администратора со всеми бронированиями читателей. Возможность
изменения каждой брони. Видны даты, когда читатель взял книгу, возможен переход
на страницу читателя.

### 3. Readers
Страница со списком читетелей по логинам - ссылками на личную страницу
читателей.

### 4. Add book
Кнопка, по которой осуществляется переход на страницу заполнения информации о
новой книге.

### 5. Add reader
Кнопка добавления читателя - появляется страница для заполнения информации о
читателе.

### 6. Log out
Переход на главную страницу для незарегистрированного пользователя.

Сценарии использования
-----------
### 1. Незарегистрированные пользователи
 - Просмотр списка всех книг на главной странице.
 - - с любой страницы - кнопка "Main page" -> переход на главную страницу со
     списком книг.
 - Возможность зарегистрироваться на сайте на странице "Регистрация".
 - - С любой страницы: кнопка "Регистрации" в верхнем меню -> переход на
     страницу "Регистрация".

### 2. Зарегистрированные пользователи
 - Просмотр список всех книг на главной странице
 - - С любой страницы: кнопка "Main page" в верхнем меню -> переход к списку
     книг.
 - - С любой страницы: кнопка "Личный кабинет" в верхнем меню -> переход на страницу личного кабинета.
 - Редактирование Личного кабинета
 - - С любой страницы: кнопка "Личный кабинет" в верхнем меню -> переход на страницу личного кабинета -> кнопка "Редактировать" -> переход на страницу редактирования личного кабинета
 - Возможность добавлять книги в определенном количестве в бронирования
 - - С любой страницы: кнопка "Main page" в верхнем меню -> переход на страницу со списком книг -> Кнопка "Забронировать книгу" -> переход на страницу редактирования количества добавляемых книг
 - Просмотр бронирований, изменение количества книг в ней.
 - - С любой страницы: кнопка "Бронирования" -> переход на страницу "Бронирования" -> кнопка "Редактировать" -> переход на страницу редактирования страницы.
 - Переход на страницу "Оформление бронирования"
 - - С любой страницы: кнопка "Main page" -> переход на страницу со списком книг -> Переход на страницу книги -> Кнопка "Забронировать"

### 3. Администраторы
 - Просмотр всех бронирований от пользователей
 - - С любой страницы: кнопка "Бронирования" -> переход на страницу
     "Бронирования"
 - Редактирование броней
 - - С любой страницы: кнопка "Бронирования" -> переход на страницу "Бронирования" -> кнопка "Редактировать" -> переход на страницу редактирования бронирований.
 - Просмотр личных кабинетов читателей
 - - С любой страницы: кнопка "Читатели" -> переход на страницу "Читатели" -> переход на страницу личного кабинета читателя по нажатию на пользователя -> кнопка "Редактировать" -> переход на страницу редактирования страницы.
 - Добавление новых книг
 - - С любой страницы: кнопка "Main page" в верхнем меню -> переход на страницу со списком книг -> кнопка "Добавить книгу" -> переход на страницу добавления книги
 - Удаление книг
 - - С любой страницы: кнопка "Main page" в верхнем меню -> переход на страницу со списком книг -> кнопка "Удалить книгу"
 - Редактирование информации о книгах
 - - С любой страницы: кнопка "Main page" в верхнем меню -> переход на страницу со списком книг -> кнопка "Редактировать" -> переход на страницу редактирования книг.
